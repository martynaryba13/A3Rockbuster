# A3Rockbuster
Achievement 3 Rockbuster Stealth Data Analysis Project (CareerFoundry)

## Objective
The Rockbuster Stealth Management Board has asked a series of business questions and they expect data-driven answers that they can use for their 2020 company strategy. Here are the main questions they’d like to answer:
  1. Which movies contributed the most/least to revenue gain?
  2. What was the average rental duration for all videos?
  3. Which countries are Rockbuster customers based in?
  4. Where are customers with a high lifetime value based?
  5. Do sales figures vary between geographic regions?

## Data
In this project, we are using a data set that contains information about Rockbuster’s film i nventory, customers, and payments, among other things.

## Tools
For this project we are using SQL to analyse the data set.
